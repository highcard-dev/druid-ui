# Simple Example (WASM Sandbox)

Basic Druid UI example running in WASM sandbox mode.

## Features

- ✅ WASM sandboxing for security
- ✅ Component composition (functional & class-based)
- ✅ Event handling with state
- ✅ Client-side routing
- ✅ Hot Module Replacement

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

The component is compiled to WebAssembly (`simple.wasm`) for secure execution.
